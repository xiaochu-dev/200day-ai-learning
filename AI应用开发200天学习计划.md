## Phase 1：基础夯实（Day 1-40，80h AI + 20h 英语）

**目标**：Python从"考试会写"到"能独立写程序"；英语词汇量打底。

### 英语（每天30min）

| 内容 | 方法 | 时间 |
|------|------|------|
| 背单词 | 百词斩/墨墨，每天30个新词+复习，目标累计1500词 | 15min |
| 听力 | 每天1篇VOA常速/六级真题听力，精听（听不懂的倒回去） | 15min |

> **关键**：单词不要背拼写，认识意思+能听懂发音就够了。六级考的是识别，不是拼写。

### AI开发（每天90min）

| 周 | 内容 | 具体任务 |
|----|------|---------|
| W1 | Python复习+深化 | 列表推导式、dict/set、函数、异常处理、文件IO —— 每个知识点写完代码跑通 |
| W2 | 面向对象+模块 | 类/继承/装饰器/生成器/import机制 —— 写一个简单的学生管理系统（纯Python，不用框架） |
| W3 | 数据结构和算法入门 | 数组/链表/栈/队列/哈希表 —— LeetCode简单题每天1道 |
| W4 | 数据库+SQL | SQLite → 建表/CRUD/JOIN/索引。用Python操作数据库（sqlite3模块） |
| W5 | Git | init/add/commit/branch/merge/push —— 把你写的代码全部推到GitHub |
| W6 | Linux基础 | 文件操作/权限/进程/管道/grep —— WSL或虚拟机里敲，别只看 |

**Phase 1 检查点**（Day 40）：
- [ ] GitHub绿墙连续40天
- [ ] 能不看资料写出一个带数据库的Python CLI程序
- [ ] LeetCode简单题做过30道
- [ ] 六级词汇累计背了1200个

## Phase 2：AI核心技能（Day 41-90，100h AI + 25h 英语）

**目标**：能调用LLM API，写出第一个AI应用后端。

### 英语（每天30min）

| 内容 | 方法 |
|------|------|
| 单词继续 | 累计到3000词 |
| 阅读 | 每天1篇六级阅读真题，计时做，做完精读 |
| 听力 | 继续每天1篇精听 |

### AI开发（每天90min）

| 周 | 内容 | 具体任务 |
|----|------|---------|
| W7-8 | FastAPI | 路由/请求体/参数校验/异常处理/中间件 —— 做一个RESTful API服务 |
| W9-10 | LLM API调用 | DeepSeek API/OpenAI兼容接口 —— 同步/异步/流式输出/错误重试/速率限制 |
| W11 | Prompt Engineering | System prompt设计/few-shot/Chain-of-Thought/输出格式约束(JSON mode) —— **这是核心能力** |
| W12-13 | 向量数据库 | 嵌入(embedding)概念 → ChromaDB → 文本切块策略（按语义、固定窗口、递归） → 相似度检索 |
| 缓冲周 | 查漏补缺 | 前面没消化的内容，这周补上 |

**Phase 2 关键产出**：
- 一个能跑的FastAPI服务（有路由、有数据库、有错误处理）
- 一段能稳定调通DeepSeek API的代码（含流式输出和异常处理）

## 每天2小时怎么安排

```
30min  英语（雷打不动，放在早上或睡前）
        ├── 15min 单词
        └── 15min 听力/阅读/真题

90min  AI开发（放在精力最好的时间段）
        ├── 15min 复习昨天学的
        ├── 60min 新课/写代码
        └── 15min 记笔记/提交GitHub
```

---

## 三个硬性规则

1. **GitHub每天至少1次commit**。不写代码也要更新笔记。绿墙是面试官对你自律性的第一印象。

2. **不动手不算学**。看视频看文档不算有效时间。手敲键盘才算。

3. **周日复盘**。每周日用30分钟回顾这周学了什么、哪些没消化、下周重点是什么。

---

## 风险预案

| 风险 | 应对 |
|------|------|
| 某天太累学不动 | 最少保证30min英语+1个GitHub commit。可以只复习不学新东西。断一天容易断一周 |
| 项目卡住了 | 2小时内解决不了就记下来，第二天继续。不要死磕到放弃 |
| CET-6考不过 | 考前2周英语时间翻倍。六级不是能力考试，是熟练度考试，10套真题反复做一定能过 |
| 方向不喜欢了 | Phase 2结束时（Day 90）做一次评估，如果真的不喜欢AI方向，换Go后端还来得及 |
