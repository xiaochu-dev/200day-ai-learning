记忆技巧（助记）
============================================
- commission / commitment / committee → 都来自 commit（委托/承诺/提交）
- compatible（兼容）← com(一起) + pati(承受) → 能一起承受的
- compensate（补偿）← com(一起) + pens(称量) → 称量平衡 → 补偿
- component / compose / compound / comprise → 都来自 com(一起) + ponere(放)
- conceive（构思）← con(一起) + ceive(拿) → 脑子里拿在一起 → 构思
- condemn（谴责）← con(加强) + demn(伤害) → 强烈伤害 → 谴责
- condense（浓缩）← con(一起) + dense(密集) → 更密集 → 浓缩
- confess（坦白）← con(完全) + fess(说) → 全说出来 → 坦白
